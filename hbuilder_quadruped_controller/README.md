# 四足控制器 App 使用说明

## 一、应用简介

四足控制器 App 是一款用于移动端控制四足机器人设备的 Android 应用。应用通过无线网络连接控制网关，实现机器人连接、姿态控制、运动控制、动作执行、状态查看以及视觉画面查看等功能。

本应用适合在机器人调试、演示和日常控制场景中使用。使用前请确保机器人本体、控制网关和手机处于可通信状态。

## 二、主要功能

1. 连接控制网关

   App 启动后会进入连接页面。用户可以输入或确认控制器地址，点击连接后，App 会尝试与控制网关建立通信。

2. 机器人基础控制

   连接成功后，可在控制页面执行常用控制操作，包括站立、趴下、回零、停止、急停等。

3. 运动控制

   控制页面提供双摇杆操作，用于控制机器人前进、后退、转向和横移等运动。

4. 动作执行

   App 支持发送预设动作指令，用于触发机器人执行指定动作。

5. 状态查看

   App 会显示当前连接状态、控制状态和机器人反馈信息，方便用户判断设备是否正常通信。

6. 视觉查看

   视觉页面支持查看相机画面，并可在 RGB 画面和点云显示之间切换。点云页面支持拖动旋转、缩放和复位。

## 三、使用前准备

1. 确保机器人设备已上电并完成启动。

2. 确保控制网关服务已启动。

3. 确保手机与控制网关处于同一网络环境，或二者之间网络可互通。

4. 确认控制器地址和端口正确。默认地址示例：

   ```text
   10.0.50.236:9001
   ```

   实际使用时，请根据现场网络环境填写真实地址。

## 四、基本使用流程

1. 安装并打开 App。

2. 在连接页面确认控制器地址。

3. 点击“连接”。

4. 等待连接成功。连接成功后，App 会进入控制页面。

5. 在控制页面执行站立、趴下、回零、动作、摇杆控制等操作。

6. 如需查看视觉画面，点击页面中的“视觉”入口，进入视觉页面。

7. 使用结束后，建议先停止机器人运动，再退出 App。

## 五、控制器地址修改

如果网络环境变化，控制网关地址可能会改变。此时可以在 App 的设置入口修改控制器地址。

修改后点击保存并连接，App 会使用新的地址重新建立通信。

地址格式通常为：

```text
IP地址:端口号
```

示例：

```text
192.168.1.100:9001
```

## 六、视觉功能说明

视觉页面提供两种查看模式：

1. RGB

   用于查看普通相机画面。

2. 点云

   用于查看三维点云数据。点云页面支持拖动旋转、缩放和复位操作。

注意：只有停留在点云页面时，控制网关才会持续生成点云数据。切换到 RGB 页面、返回控制页面或将 App 切到后台后，点云采集会暂停。

## 七、注意事项

1. 操作机器人前，请确保周围环境安全，避免机器人运动时碰撞人员或物品。

2. 首次使用或网络变化后，请先确认控制器地址是否正确。

3. 如果连接失败，请检查手机网络、控制网关状态、地址端口以及机器人是否正常启动。

4. 使用摇杆控制前，建议先让机器人进入稳定站立状态。

5. 发现机器人姿态异常、动作异常或通信异常时，请立即点击停止或急停。

6. App 进入后台后，部分控制和视觉数据可能会暂停刷新，继续使用时请重新确认连接状态。

7. 请不要在低电量、网络不稳定或人员密集环境下进行高风险动作测试。

## 八、常见问题

### 1. 打开 App 后无法连接

请确认手机与控制网关网络可互通，控制器地址填写正确，控制网关服务已启动。

### 2. 连接成功但机器人没有动作

请确认机器人本体已启动完成，控制服务正常运行，并检查是否处于急停或保护状态。

### 3. 视觉页面没有画面

请确认相机设备工作正常，网络连接稳定，并检查当前是否进入了正确的视觉模式。

### 4. 点云不刷新

请确认当前停留在点云页面。切换到其他页面或 App 进入后台后，点云采集会暂停。

### 5. 控制延迟明显

请检查无线网络质量，尽量减少网络拥堵，并保持手机与控制网关之间的连接稳定。
